# スマホでサイトを表示する方法

## 手順

### 1. MacとスマホでWi-FiをONにして同じネットワークに接続

Macとスマホが同じWi-Fiネットワークに接続されていることを確認してください。

### 2. ローカルサーバーを起動

ターミナルでサイトのディレクトリに移動して、以下のコマンドを実行：

```bash
cd /Users/rionaaigbe/.gemini/antigravity/playground/empty-orion/t-secondhands-site
python3 -m http.server 8000
```

サーバーが起動したら、以下のように表示されます：
```
Serving HTTP on 0.0.0.0 port 8000 (http://0.0.0.0:8000/) ...
```

### 3. MacのIPアドレスを確認

**新しいターミナルウィンドウ**を開いて、以下のコマンドを実行：

```bash
ipconfig getifaddr en0
```

例：`192.168.1.5` のような数字が表示されます。

### 4. スマホでアクセス

スマホのブラウザ（Safari/Chrome）で以下のURLにアクセス：

```
http://[上記のIPアドレス]:8000/index.html
```

例：`http://192.168.1.5:8000/index.html`

## トラブルシューティング

### IPアドレスが表示されない場合

```bash
ifconfig | grep "inet " | grep -v 127.0.0.1
```

このコマンドでIPアドレス一覧が表示されます。`192.168.`または`10.`で始まるアドレスを使用してください。

### スマホからアクセスできない場合

1. Macとスマホが同じWi-Fiに接続されているか確認
2. Macのファイアウォール設定を確認（システム環境設定 → セキュリティとプライバシー → ファイアウォール）
3. ポート8000が使用中の場合は別のポート（例：8080、3000）を使用

### サーバーを停止する方法

ターミナルで `Ctrl + C` を押してください。

## 各ページへのアクセス

- トップページ: `http://[IPアドレス]:8000/index.html`
- 商品一覧: `http://[IPアドレス]:8000/products.html`
- 商品詳細: `http://[IPアドレス]:8000/product-detail.html?id=1`
